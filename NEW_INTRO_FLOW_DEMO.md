## ğŸ¯ Why This Flow Works

### Psychological Principles:

**1. Foot-in-the-Door Effect**
- Start with small ask (Curious/Skip)
- Gradually increase commitment
- Each yes makes next yes easier

**2. Scarcity + Selectivity**
- "I'll only move forward if..."
- Not everyone gets matched
- Increases perceived value

**3. Investment Escalation**
- Answering questions = investment
- Sunk cost makes declining harder
- More invested = more likely to engage

**4. Social Proof + Authority**
- "When this works, it tends to..."
- System has judgment
- Creates trust in matching quality

**5. Loss Aversion**
- After building up value, "pass for now" feels like loss
- FOMO kicks in
- But graceful exit preserves agency
